# 🤖 AI-Powered Smart Data Analyst

🚀 An advanced **AI-driven data analysis web app** built using **Streamlit** that allows users to upload datasets, clean data, visualize insights, detect outliers, and generate professional reports.

---

## ✨ Features

* 📊 **Dashboard Overview**
* 🧹 **Data Cleaning (Nulls, Duplicates)**
* 🧠 **AI Insights Generation**
* 📈 **Advanced Visualizations (Seaborn)**
* 📉 **Trend Analysis**
* 🚨 **Outlier Detection**
* 🏆 **Data Quality Score**
* 📄 **Automated PDF Reports**
* 🤖 **AI Chat with Dataset**
* 📊 **YData Profiling Integration**

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 🎯
* Pandas & NumPy 📊
* Seaborn & Matplotlib 📈
* YData Profiling 📄

## 🚀 How to Run Locally

```bash
git clone https://github.com/your-username/AI-Data-Analyst.git
cd AI-Data-Analyst
pip install -r requirements.txt
streamlit run app.py
```
